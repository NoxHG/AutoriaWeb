# ✨ AutoriaWeb

**Meu espaço de estudos da disciplina Autoria Web** 💻

Repositório dedicado aos meus aprendizados em **HTML5**, **CSS3** e **JavaScript** puro durante a matéria de Autoria Web.

## 📌 Sobre

Este repositório reúne todos os exercícios, atividades e projetos práticos desenvolvidos ao longo do curso. Aqui eu coloco em prática os conceitos de:

- HTML5 semântico
- CSS3 moderno (Flexbox, Grid, animações, variáveis, responsividade)
- JavaScript Vanilla (DOM, eventos, manipulação de dados e interatividade)
